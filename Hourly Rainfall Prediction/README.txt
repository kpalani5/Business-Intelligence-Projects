
While executing copy all files in all folders to a single folder.

EXECUTION

1. Comparing.m

   Select a result from any one of the algorithms and any sample size/ number of trees present in the matlab data files.
   Run the comparing.m file to get the evaluation metric values.

2. Main.m

   Run the Main.m file to execute SPRINT for a particular sample size.

3. RandomForests.m

   Run RandomForests.m file to execute Random Forest classification for a particular number of trees.

4. RandomForestsRegression.m

   Run RandomForestsRegression.m to execute Random Forest regression for a particular number of trees.

5. AODE.m

   Run AODE.m to execute AODE for a particular sample size.
